Evidence-based biological knowledge graph linking Vitex agnus-castus compounds, receptors, and PMS-related physiological effects.

# Vitex × PMS – Biological Knowledge Graph

This project models the biological mechanisms of *Vitex agnus-castus* in the context of premenstrual syndrome (PMS).

## Scope
- Active compounds
- Biological receptors
- Hormonal pathways
- Physiological effects related to PMS

## Data sources
- PubMed
- PubChem

## Tech stack
- PostgreSQL (Supabase)
- SQL
- (Future) n8n / AI-based literature extraction
